# Lurnite

A web-based study management platform to help students organize tasks, schedules, and study progress in one place.

> 🚧 **Status:** Work in progress

---

## ✨ Features

* Subject and task organization
* Study scheduling
* Progress tracking
* Clean, minimal interface


## 🎯 Goals

* Improve study organization
* Encourage consistent study habits
* Serve as a practical learning project
